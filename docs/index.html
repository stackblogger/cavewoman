Open source CLI

cavewoman — universal AI response optimizer

One command installs concise, structured, low-fluff rules into Cursor skills, Claude Code plugins, ChatGPT clipboard, Codex wrappers, Windsurf prefixes, and more. Built for real terminals: monospace type, fast defaults, agent-specific injectors.

Install

Global install or run on demand with npx (no global install).

zsh — 80×24
$ npm install -g cavewoman
$ npx cavewoman --target cursor

Quick start

Omit flags on install for an interactive TTY picker (↑/↓, Enter or Space). Otherwise pass --target, --mode, and --scope.

cavewoman
$ cavewoman install --target cursor --mode structured --scope global
$ cavewoman install --target claude-code --mode ultra
$ cavewoman status
$ cavewoman switch balanced
$ cavewoman uninstall --target cursor
done
cavewoman setup
✔ Installed cavewoman skill
cavewoman is now active.

Supported agents

Target Behavior
cursor Skill at ~/.agents/skills/cavewoman (global) or ./.agents/skills/cavewoman (project)
chatgpt Sticky file under ~/.cavewoman/chatgpt-sticky-prompt.txt plus clipboard copy (clipboardy).
claude-code Claude Code plugin layout under ~/.cavewoman/claude-code-plugin (global) or ./.cavewoman/claude-code-plugin (project): .claude-plugin/plugin.json + skills/cavewoman/SKILL.md. Load with claude --plugin-dir … or publish via a plugin marketplace.
gemini Gemini CLI extension: gemini-extension.json + GEMINI.md under ~/.gemini/extensions/cavewoman (global) or ./.gemini/extensions/cavewoman (project). Restart the CLI; install from disk with gemini extensions link or from Git with gemini extensions install.
codex Prefix file + cavewoman-codex wrapper under ~/.cavewoman/bin (adjust if your Codex CLI flags differ)
windsurf windsurf-prefix.txt for Cascade-style memory
generic generic-prefix.txt to prepend manually

Modes

  • balanced — tighter prose, fewer headings
  • structured — headings + checklist-style contract
  • ultra — minimum tokens, imperative bullets

Configuration

State lives in ~/.cavewomanrc (defaults, last target, scope, preferences). Optional CAVEWOMAN_CURSOR_SKILLS_DIR overrides the Cursor skills root without writing to the rc file.
